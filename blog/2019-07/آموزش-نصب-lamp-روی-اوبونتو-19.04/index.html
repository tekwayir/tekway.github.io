آموزش نصب LAMP روی اوبونتو 19.04
  • Jul 11, 2019
  • 3 دقیقه برای خواندن

لمپ (LAMP) مجموعه‌ای از نرم‌افزارهای آزاد و متن باز که برای میزبانی محتوای وب در محیط لینوکس استفاده میشه. تو این آموزش به شما نشون میدیم که چجوری لمپ رو روی اوبونتو ۱۹.۰۴ نصب کنید.

گام اول: آپدیت کردن پکیج‌های اوبونتو

خب قبل از نصب لمپ بهتره که مخازن و بسته‌های نرم‌افزاری اوبونتو رو آپدیت کنید. برای این کار از دستورات زیر استفاده میکنیم:

sudo apt update
sudo apt upgrade

گام دوم: نصب آپاچی (Apache Web Server)

با دستورات زیر آپاچی رو نصب کنید. بسته‌ی apache2-utils برخی ابزارهای مفید مثل Apache HTTP Server Benchmarking tool رو نصب خواهد کرد.

sudo apt install -y apache2 apache2-utils

بعد از نصب، آپاچی به صورت خودکار شروع به کار خواهد کرد. وضعیت آپاچی رو با دستور زیر چک کنید.

systemctl status apache2

اگر که خروجی به صورت بالا نبود، از ‍‍‍systemctl برای اجرا شدن آپاچی استفاده کنید:

sudo systemctl start apache2

همچنین برای اجرا شدن آپاچی بعد از روشن شدن سیستم بهتره که آپاچی رو فعال کنید:

sudo systemctl enable apache2

با دستور زیر میتونید ورژن آپاچی رو چک کنید:

apache2 -v

خروجی به صورت زیر است:

Server version: Apache/2.4.38 (Ubuntu)
Server built:   2019-04-03T18:31:46

حالا برای اطمینان از اجرا شدن آپاچی توی مرورگر خودتون یکی از آدرس های 127.0.0.1 یا localhost رو وارد کنید. خروجی باید به صورت زیر باشه:

اگر درخواست رد شد یا انجام نشد، ممکنه که iptables فایروال از درخواست‌های ورودی پورت 80 جلوگیری میکند. برای باز کردن پورت TCP 80 باید دستور زیر رو اجرا کنید:

sudo iptables -I INPUT -p tcp --dport 80 -j ACCEPT

اگر شما از UFW فایروال استفاده میکنید دستور زیر را برای بازکردن پورت TCP 80 استفاده کنید:

sudo ufw allow http

خب حالا ما نیاز داریم که www-data رو هم به عنوان document root تنظیم کنیم.

sudo chown www-data:www-data /var/www/html/ -R

گام سوم: نصب MariaDB Database Server

MariaDB یک جایگزین برای MySQL هستش.

برای نصب روی اوبونتو 19.04 دستور زیر رو توی ترمینال وارد کنید:

sudo apt install mariadb-server mariadb-client

بعد از نصب MariaDB خودکار راه‌اندازی میشه. برای چک کردن وضعیت از دستور زیر استفاده میکنیم:

systemctl status mariadb

اگر که برای شما به درستی کار نکرد دستور زیر رو وارد کنید:

sudo systemctl start mariadb

برای فعال کردن اجرای خودکار هنگام روشن کردن دستگاه، از دستور زیر استفاده کنید:

sudo mysql_secure_installation
